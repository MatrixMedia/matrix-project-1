Events

Show Public Events
The first toggle determines whether Events tagged public will be shown, the second toggle determines the same for events tagged For Members Only, and the third toggle determines whether past events will be shown.
Show Member’s only Events
You may use this toggle to choose whether your Community’s events will be shown on the public “Events” page and the public “Home” page.
Show Past Events
Note that Past events will be shown only if the corresponding first two toggles are set to show such events (Public and Members Only).
